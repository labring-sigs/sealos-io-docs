# components

## 目的
沉淀可复用的 UI 组件与页面片段。

## 模块概述
- **职责:** 组件实现、样式与交互复用
- **状态:** ✅稳定
- **最后更新:** 2026-01-20

## 规范

### 需求: 基础维护
**模块:** components
组件需保持语义清晰与复用一致。

#### 场景: 日常更新
新增或调整组件时保持接口稳定。
- 使用方不受破坏性影响
- 组件表现符合设计

### 需求: MDX 图片组件
**模块:** components
提供 MDX 图片组件以支持运行时域名替换。

#### 场景: 图片地址重写
在渲染时替换 MDX 图片地址为当前部署域名。
- 兼容现有 CDN 图片地址
- 兼容非字符串 src（如静态导入）
- 保持图片圆角样式
