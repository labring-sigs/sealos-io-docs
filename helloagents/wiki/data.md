# 数据模型

## 概述
本仓库以静态内容文件为主，未维护独立数据库或服务端数据模型。

---

## 数据表/集合

### 内容文件

**描述:** 文档内容与参考资料（MDX/Markdown/JSON）。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| slug | string | 唯一 | 内容路径标识 |
| title | string | 可选 | 标题或显示名称 |
| locale | string | 可选 | 语言标识 |
| body | string | 可选 | 正文内容 |

**索引:**
- slug

**关联关系:**
- 内容可被页面模块引用
