# 项目技术约定

---

## 技术栈
- **核心:** Node.js + Next.js (App Router) + React + TypeScript
- **样式:** PostCSS（如有其他样式体系请补充）

## 开发约定
- **包管理:** npm（存在 `package-lock.json`）

- **命名约定:** 目录使用短小语义名；组件与类型遵循 TypeScript 习惯（如有其他约定请补充）

---
