欢迎使用 DevBox！本指南带你快速掌握从项目创建到应用上线的完整开发流程。

## 核心组件
DevBox 提供了一套完整的云端开发到部署解决方案，包含以下核心组件：

| 组件 | 描述 |
|-----------|-------------|
| 项目列表 | 统一管理多个应用项目，每个项目运行在独立容器中，实现资源隔离和环境隔离 |
| 开发环境 | 开箱即用的环境模板，内置所有语言框架，支持VS Code 等所有IDE连接，随时随地编码 |
| 发布版本 | 自动将应用打包为标准 OCI 镜像，支持版本管理和快速回滚 |
| 部署上线 | 无缝对接【应用管理】，实现应用快速上线和弹性伸缩 |

接下来，我们将通过以下几个章节，详细介绍如何使用这些核心组件，帮助你构建一个完整的开发工作流：

## 开发工作流

### 1️⃣ 新建项目

- 创建项目并选择合适的运行环境
- 配置项目基础信息和资源规格
- 每个项目独立容器隔离运行

**详细文档** → [新建项目指南](./create-a-project)

---

### 2️⃣ 开发调试

- 连接云端开发环境
- 使用 Vscode、VS Code 等熟悉的 IDE 进行开发
- 实时预览和调试应用

**详细文档** → [开发调试指南](./develop)

---

### 3️⃣ 配置启动

- 设置应用启动命令和参数
- 配置环境变量和端口映射
- 定义健康检查规则

**详细文档** → [启动配置指南](./entrypoint-sh)

---

### 4️⃣ 发布版本

- 自动构建并发布 OCI 标准镜像
- 管理应用版本历史
- 支持版本回滚和对比

**详细文档** → [版本发布指南](./release)

---

### 5️⃣ 部署上线

- 一键部署到 Sealos Cloud
- 配置自定义域名和 SSL 证书
- 设置访问策略和流量控制

**详细文档** → [部署上线指南](./deploy)

---

💡 **提示**：完整的开发流程只需 5 步，从创建项目到应用上线，全程可视化操作，无需复杂的配置。

通过学习这些基础内容，你将能够熟练运用 Sealos DevBox 的各项功能，最初的项目创建，到最终的应用部署，让开发过程更加顺畅。
