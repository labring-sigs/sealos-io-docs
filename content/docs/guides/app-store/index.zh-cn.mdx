title: 应用商店
description: 应用商店是 Sealos 云操作系统中的一个功能，用于管理应用的安装、更新、删除等操作。
---

提供了一系列预制的应用模板，支持快速创建和部署各种网站及应用程序。这些模板包括博客、AI 应用、低代码应用、网盘、即时通讯应用、中间件等，旨在简化开发过程，让开发者无需从零开始构建项目或处理应用之间的依赖关系。

![Sealos 应用商店](./images/sealos-app-store.zh-cn.png)

## 提交应用

如果您想向 Sealos 应用商店提交应用，可以直接提交 PR (Pull Request)。

如果您不熟悉如何编写应用的 YAML 模板，也可以提交一个新的 Issue，描述您想要添加的应用。我们的维护团队会协助您完成应用模板的编写和提交。在提交 Issue 时，请尽可能详细地说明：

- 应用的名称和简介
- 应用的官方网站或 GitHub 仓库地址
- 应用的部署要求 (如：所需的环境变量、存储需求等)
- 其他相关信息

这样可以帮助我们更好地理解您的需求，加快应用模板的制作过程。
