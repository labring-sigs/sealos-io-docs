欢迎使用 Sealos DevBox！本指南带你快速掌握从项目创建到应用上线的完整开发流程。

## 核心组件
DevBox 提供了一套完整的云端开发到部署解决方案，包含以下核心组件：

| 组件 | 描述 |
|-----------|-------------|
| 项目列表 | 统一管理多个应用项目，每个项目运行在独立容器中，实现资源隔离和环境隔离 |
| 开发环境 | 开箱即用的环境模板，内置所有语言框架，支持VS Code 等所有IDE连接，随时随地编码 |
| 发布版本 | 自动将应用打包为标准 OCI 镜像，支持版本管理和快速回滚 |
| 部署上线 | 无缝对接【应用管理】，实现应用快速上线和弹性伸缩 |

接下来，我们将通过以下几个章节，详细介绍如何使用这些核心组件，帮助你构建一个完整的开发工作流：

1. [新建项目](./guides/fundamentals/create-a-project)：手把手教你在 Sealos DevBox 中创建和配置新项目。
2. [开发调试](./guides/fundamentals/develop)：学习如何连接开发环境并开始编码工作。
3. [应用启动配置](./guides/fundamentals/entrypoint-sh)：了解如何配置应用的启动命令。
4. [发布版本](./guides/fundamentals/release)：掌握如何将应用打包为标准 OCI 镜像并进行版本管理。
5. [部署上线](./guides/fundamentals/deploy)：了解如何将应用快速部署到云端并对外提供服务。

通过学习这些基础内容，你将能够熟练运用 Sealos DevBox 的各项功能，打造一个高效的开发工作流程。从最初的项目创建，到最终的应用部署，每一步都经过精心设计，让开发过程更加顺畅。
