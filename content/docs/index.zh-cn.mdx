## 什么是 Sealos？

Sealos 是一个以 Kubernetes 为内核的云操作系统，提供了云原生应用的完整生命周期管理能力。它将复杂的云基础设施抽象为简单易用的应用，让开发者可以像使用个人电脑一样使用云计算资源。

## 系统概览

Sealos 采用模块化设计，每个功能都以独立应用的形式提供，用户可以根据需要选择使用。所有应用都基于 Kubernetes 构建，共享底层资源池，实现资源的高效利用和统一管理。

## 核心应用

### 🔧 DevBox - 云端开发环境
- 提供开箱即用的云端开发环境
- 支持所有主流开发语言和框架（Node.js、Python、Go、Java 等）
- 集成 Vscode、VS Code 等 IDE，支持本地连接开发
- 内置版本管理，自动构建 OCI 镜像
- 一键部署到生产环境

**适用**
- 团队协作开发，统一开发环境
- 快速原型验证和演示

---

### 🚀 应用管理 - 容器应用部署
- 支持 Docker 镜像一键部署
- 可视化配置 CPU、内存、副本数
- 自动负载均衡和健康检查
- 支持水平自动扩缩容（HPA）
- 域名绑定和 SSL 证书管理
- 应用监控和日志查看

**适用**
- Web 应用部署和托管
- API 服务部署

---

### 💾 数据库 - 托管数据库服务
- 支持多种数据库类型：
  - 关系型：MySQL、PostgreSQL
  - 文档型：MongoDB
  - 键值型：Redis
- 一键创建，自动配置高可用
- 自动备份和恢复
- 性能监控和慢查询分析
- 可视化数据管理工具

**适用**
- 应用数据持久化存储
- 缓存和会话管理
- 数据分析和报表
- 日志和监控数据存储

---

- 支持大文件上传和断点续传
- 图片、视频等多媒体文件存储
- 访问权限控制（公开/私有）
- CDN 加速支持
- 存储空间和流量统计

**适用**
- 静态资源托管
- 用户上传文件存储
